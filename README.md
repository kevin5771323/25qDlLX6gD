## 前言

您好！欢迎访问我们的“高校学术研讨信息管理系统+SpringBoot”项目。该项目旨在为广大高校师生提供一个便捷、高效的学术研讨信息管理平台，通过整合各类资源，实现研讨信息的实时发布、查询与管理。以下是项目相关内容的详细介绍。

## 内容介绍

“高校学术研讨信息管理系统+SpringBoot”项目是一个基于Java语言的Web应用，采用Spring、SpringMVC、MyBatis等主流框架进行开发。系统前端采用JS、Vue、CSS3等技术实现，同时支持微信小程序和Uniapp跨平台应用。通过该项目，用户可以轻松发布和参与学术研讨活动，提高学术交流的效率。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
